<h1>Forwarder Bot</h1>
<b>What is this ?</b>
<p>a small telegram bot for forwarding files / media / message to selected channel / groups[?threads] </p>


<b>How to run bot ?</b>
<p>Simply deploy use docker container</p>

<b>Bot command :</b><br>
<code>/start</code> -> show all commands that exist.<br>
<code>/setchat <chat_id></code> -> set target chat id to bot.<br>
<code>/sendhere </code> -> set target to current chat, useful if you dont know the id, just run target chat where bot is existed, ex. a group.<br>
<code>/check </code> -> check if bot can send a message to target.<br>
<code>/status </code> -> show current target that is being setted.<br>
<code>/alwaysdl <0/1> </code> -> 1 reupload function, 0 is telegram native forwarder<br>
<code>/queue </code> -> show a current queue.<br>
